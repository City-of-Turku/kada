== PURPOSE ==
Has all user related configuration like roles and strongarm variables.
All permissions which are not related to other features are in this feature.

== ROLES ==
- admin: Site builders and developers. People who are not afraid to touch Drupal configuration.
- content editor: More restricted role, which has less access than web journalist in many ways.
- web journalist: Can access more or less all content and trickier configuration than content editors.

None.
